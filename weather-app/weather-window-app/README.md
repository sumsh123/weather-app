# Weather Window Clean

This version removes the three large decorative weather shapes from the main scene.

The main screen now shows:

- Temperature
- Condition
- Feels-like temperature
- Humidity
- Wind
- Pressure
- Visibility

The details popup shows:

- Pressure
- Visibility
- Sunrise
- Sunset
- Local time
- Update time
- Four forecast periods
- Celsius/Fahrenheit switch

## Add the API key

Open `config.js` and replace the placeholder with your real OpenWeather API key.

## Run

Open `index.html` with Live Server.
